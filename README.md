# 📊 Student Performance Analysis Dashboard

## 📌 Project Overview

This Power BI dashboard analyzes student academic performance using various factors such as gender, ethnicity, parental education, and test preparation. The dashboard provides interactive insights into students' scores and helps identify performance trends.

---

## 🎯 Objectives

- Analyze student academic performance.
- Compare average scores across different categories.
- Identify the impact of parental education.
- Evaluate the effect of test preparation on student scores.
- Build an interactive dashboard using Power BI.

---

## 📂 Dataset Information

The dataset contains information about students, including:

- Student ID
- Gender
- Ethnicity
- Parental Education
- Test Preparation
- Math Score
- Reading Score
- Writing Score

---

## 📈 Dashboard Features

### KPI Cards
- 👨‍🎓 Student Count
- ➗ Average Math Score
- 📖 Average Reading Score
- ✍️ Average Writing Score
- 📊 Overall Average Score

### Visualizations
- Average Score by Ethnicity
- Average Score by Gender
- Average Score by Parental Education
- Average Score by Test Preparation
- Top 5 Student Performance
- Ethnicity vs Parental Education Analysis

### Interactive Filters
- Ethnicity
- Gender
- Parental Education
- Test Preparation
