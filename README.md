# Student Performance Analysis Dashboard

## Project Overview
This project analyzes student academic performance using Power BI.

## Tools Used
- Power BI Desktop
- Power Query
- CSV Dataset

## Dashboard Features
- Total Students KPI
- Average Final Grade
- Average Absences
- Average Study Time
- Grade by Gender
- Grade by Study Time
- Grade by School
- Student Distribution
- Absences vs Final Grade
- Interactive Slicers

## Dataset
Student Performance Dataset (CSV)
